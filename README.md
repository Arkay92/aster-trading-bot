# Aster Trading Bot

Bot-only TypeScript project for running automated trading strategies on Aster futures.

## Run

```bash
npm install
npm run bot
```

## Scripts

- `npm run bot`
- `npm run test:bot`
- `npm run test:step`
- `npm run test:live:btc`
- `npm run test:live:btc:v3`
- `npm run register:api-key`

## Notes

- Frontend/Next.js code and dependencies were removed.
- Runtime logs are written under `log/`.
